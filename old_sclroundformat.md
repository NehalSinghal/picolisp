## round and format functions
```
: (round 1234 3)
-> "1.234"
: (round 1234)
-> "1.234"
: (round 12345678 5)
-> "12,345.678"
: (format 123679)
-> "123679"
: (format 1236554 *Scl)
-> "1236.554"
: (format 4566343789 6)
-> "4566.343789"
```
